# Gemini Project Instructions

This file provides instructions for the Gemini AI to interact with this project.

## Project Overview

This project is a collection of Spixi mini-apps and related tools. The main components are:

-   `apps/`: Contains the source code for the mini-apps.
-   `Go_Spixi_devPack/`: A Go project for Spixi development.
-   `mini-apps-sdk/`: The Spixi mini-app SDK.

## General Instructions

-   When modifying code, please follow the existing coding style.
-   Ensure that any new features are accompanied by tests.
-   Commit directly to the main branch.
